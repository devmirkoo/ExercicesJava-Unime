# Esercizio 5 – Simulazione della Crescita di una Popolazione (Ciclo For) / Population Growth Simulation (For Loop)

---

## 🇮🇹 Italiano

### Descrizione
Programma che simula la **crescita di una popolazione di insetti** nell'arco di 10 settimane, usando un ciclo `for`.

L'utente inserisce:
- Il **numero iniziale di insetti**
- Il **tasso di crescita settimanale** (moltiplicatore, es. `2` per raddoppio)

Il programma mostra settimana per settimana la popolazione aggiornata, moltiplicando il numero di insetti per il tasso di crescita ad ogni iterazione.

### Concetti trattati
- Ciclo `for` con numero di iterazioni noto
- Input con `Scanner`
- Aggiornamento di una variabile ad ogni iterazione
- Output formattato con `System.out.printf`

### Esempio di esecuzione
```
Inserisci il numero di insetti, e il tasso di crescita settimanale.
10
2
Settimana 1
Numero attuale insetti: 20

Settimana 2
Numero attuale insetti: 40

...

Settimana 10
Numero attuale insetti: 10240

I tuoi insetti sono diventati in totale: 10240
```

## 🇬🇧 English

### Description
A program that simulates the **growth of an insect population** over 10 weeks using a `for` loop.

The user enters:
- The **initial number of insects**
- The **weekly growth rate** (multiplier, e.g. `2` to double each week)

The program displays the updated population week by week, multiplying the insect count by the growth rate on each iteration.

### Concepts covered
- `for` loop with a known number of iterations
- Input with `Scanner`
- Updating a variable on each iteration
- Formatted output with `System.out.printf`

### Sample run
```
Inserisci il numero di insetti, e il tasso di crescita settimanale.
10
2
Settimana 1
Numero attuale insetti: 20

Settimana 2
Numero attuale insetti: 40

...

Settimana 10
Numero attuale insetti: 10240

I tuoi insetti sono diventati in totale: 10240
```
